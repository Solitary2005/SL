def visualize_2d(ip_vectors_2d: np.ndarray, labels: np.ndarray, title: str, xlabel: str, ylabel: str) -> plt.Figure:
    fig = plt.figure(figsize=(8, 6))
    plt.scatter(
        ip_vectors_2d[:,0],
        ip_vectors_2d[:,1],
        c=labels,
        cmap='viridis',
    )
    cbar = plt.colorbar()
    cbar.set_label("Class Labels")
    plt.xlabel(xlabel)
    plt.ylabel(ylabel)
    plt.title(title)

    return fig

2. 关于模型优化过程：
一般是先optimizer.zero_grad()  清除上一轮的梯度
然后loss.backward()  计算这一轮的梯度
然后 optimizer.step()  更新优化器参数

如果是梯度累积下：
则每个step都计算loss.backward，计算梯度
只当%acc_steps == 0时，才先更新优化器参数optimizer.step()，再梯度清0
